Kebijakan Privasi

Terakhir diperbarui: 9 Februari 2026

Ringkasan: PajakPas hanya mengumpulkan data yang diperlukan untuk layanan pelacakan pajak kendaraan. Kami tidak menjual data Anda kepada pihak ketiga. Kami tidak menampilkan iklan. Anda dapat menghapus akun dan semua data kapan saja.

1. Pendahuluan

PajakPas ("kami", "aplikasi") adalah aplikasi mobile yang membantu pemilik kendaraan Indonesia melacak pembayaran pajak kendaraan (PKB) dan masa berlaku STNK. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi pribadi Anda saat menggunakan aplikasi kami.

Dengan menggunakan PajakPas, Anda menyetujui pengumpulan dan penggunaan informasi sebagaimana dijelaskan dalam kebijakan ini. Jika Anda tidak setuju dengan kebijakan ini, harap jangan menggunakan aplikasi kami.

2. Informasi yang Kami Kumpulkan

a. Informasi Akun (Wajib)

Saat Anda membuat akun, kami mengumpulkan:

  • Alamat email — untuk autentikasi, pemulihan akun, dan komunikasi penting terkait layanan
  • Nama — untuk tampilan profil pengguna dalam aplikasi
  • ID Pengguna — dihasilkan secara otomatis oleh sistem untuk mengidentifikasi akun Anda

Jika Anda masuk dengan Google atau Facebook, kami menerima nama dan email Anda dari layanan tersebut. Kami tidak menerima kata sandi Anda dari layanan ini.

b. Nomor Telepon (Opsional)

Anda dapat secara opsional menambahkan nomor telepon ke profil Anda. Data ini tidak wajib untuk menggunakan aplikasi.

c. Informasi Kendaraan (Konten Buatan Pengguna)

Saat Anda menambahkan kendaraan, Anda memberikan:

  • Nomor plat, merek, model, tahun, warna kendaraan
  • VIN (Nomor Identifikasi Kendaraan) dan nomor mesin
  • Nama pemilik dan alamat
  • Tanggal berlaku STNK

d. Informasi Pembayaran Pajak (Konten Buatan Pengguna)

Saat Anda mencatat pembayaran pajak, Anda memberikan:

  • Tahun pajak, tanggal pembayaran, dan masa berlaku
  • Jumlah pajak (PKB, SWDKLLJ, biaya admin, denda)
  • Nomor kuitansi

e. Gambar Dokumen (Diproses Sementara)

Saat Anda menggunakan fitur pemindaian dokumen:

  • Foto STNK atau slip pajak SSPD diambil dari kamera atau galeri Anda
  • Gambar dikirim ke layanan pihak ketiga (Anthropic) untuk ekstraksi teks bertenaga AI (OCR)
  • Gambar diproses secara real-time dan tidak disimpan di server kami maupun server pihak ketiga
  • Hanya data teks yang diekstrak yang disimpan ke akun Anda

Penggunaan fitur ini bersifat opsional — Anda dapat memasukkan data kendaraan secara manual.

f. Data Cadangan (Diproses Sementara)

Saat Anda menggunakan fitur impor/ekspor cadangan:

  • File cadangan dibuat di perangkat Anda yang berisi data kendaraan dan pajak
  • File impor dibaca untuk memulihkan data ke akun Anda
  • File cadangan diproses secara lokal dan tidak diunggah ke server kami

g. Informasi yang TIDAK Kami Kumpulkan

PajakPas tidak mengumpulkan:

  • Data lokasi (GPS, lokasi perkiraan, atau lokasi tepat)
  • Kontak dari perangkat Anda
  • Pesan (SMS, email, atau pesan lainnya)
  • Riwayat penelusuran atau aktivitas web
  • Informasi keuangan (kartu kredit, rekening bank)
  • Informasi kesehatan atau biometrik
  • Log kerusakan atau data diagnostik
  • ID perangkat atau pengidentifikasi iklan

4. Berbagi Data dan Layanan Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak ketiga untuk tujuan komersial mereka.

Kami menggunakan layanan pihak ketiga berikut semata-mata untuk mengoperasikan aplikasi:

5. Izin Aplikasi

PajakPas meminta izin perangkat berikut:

  • Kamera — untuk memindai dokumen STNK/SSPD dengan OCR. Hanya digunakan saat Anda memilih untuk memindai.
  • Penyimpanan/Galeri Foto — untuk memilih gambar dokumen dari galeri Anda. Hanya digunakan saat Anda memilih gambar.
  • Notifikasi — untuk mengirim pengingat tenggat pajak. Anda dapat menonaktifkan ini kapan saja melalui pengaturan perangkat.

Semua izin bersifat opsional dan diminta saat dibutuhkan. Aplikasi tetap berfungsi tanpa memberikan izin ini.

6. Keamanan Data

Kami menerapkan langkah-langkah keamanan standar industri untuk melindungi data Anda:

  • Semua data ditransmisikan melalui HTTPS/TLS (enkripsi dalam transit)
  • Data disimpan dalam database terenkripsi (enkripsi saat diam)
  • Autentikasi menggunakan token JWT standar industri
  • Akses database dilindungi oleh Row-Level Security (RLS) — Anda hanya dapat mengakses data Anda sendiri
  • Kata sandi di-hash menggunakan bcrypt dan tidak pernah disimpan dalam teks biasa
  • Kunci API Supabase dibatasi untuk operasi tingkat pengguna saja

Meskipun kami berusaha melindungi data Anda, tidak ada metode transmisi atau penyimpanan elektronik yang 100% aman. Kami tidak dapat menjamin keamanan mutlak.

7. Penyimpanan dan Penghapusan Data

Penyimpanan

  • Data akun, kendaraan, dan pajak disimpan selama akun Anda aktif
  • Gambar dokumen diproses secara real-time dan tidak disimpan (retensi nol)
  • File cadangan hanya ada di perangkat Anda dan tidak disimpan di server kami

Penghapusan

  • Anda dapat menghapus kendaraan dan catatan pajak individual kapan saja melalui aplikasi
  • Anda dapat menghapus seluruh akun melalui tab Profil di aplikasi atau melalui halaman penghapusan akun
  • Saat akun dihapus, semua data dihapus secara permanen dan langsung dari server kami
  • Tidak ada masa pemulihan — setelah dihapus, data tidak dapat dipulihkan
  • Data yang dihapus meliputi: profil pengguna, semua kendaraan, semua catatan pajak, pengaturan notifikasi, dan kredensial autentikasi

8. Transfer Data Internasional

Data Anda dapat diproses di server yang berlokasi di luar Indonesia, termasuk:

  • Supabase — server di wilayah Asia Tenggara (AWS)
  • Anthropic — server di Amerika Serikat (hanya untuk pemrosesan OCR sementara)

Kami memastikan bahwa semua penyedia layanan pihak ketiga menerapkan langkah-langkah perlindungan data yang memadai.

9. Hak Anda

Anda memiliki hak-hak berikut atas data Anda:

  • Akses — semua data Anda terlihat dalam aplikasi kapan saja
  • Portabilitas — gunakan fitur Ekspor Cadangan di Profil untuk mengunduh semua data Anda
  • Penghapusan — hapus catatan individual atau seluruh akun Anda dari tab Profil, atau melalui halaman penghapusan akun
  • Koreksi — edit informasi kendaraan dan pajak kapan saja melalui aplikasi
  • Pencabutan persetujuan — Anda dapat berhenti menggunakan fitur opsional (OCR, cadangan) kapan saja

Untuk menjalankan hak Anda atau jika Anda memiliki pertanyaan, hubungi kami di support@pajakpas.com. Kami akan merespons dalam waktu 48 jam.

10. Privasi Anak-anak

PajakPas tidak ditujukan untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun. Jika Anda mengetahui bahwa seorang anak telah memberikan data pribadi kepada kami, silakan hubungi kami dan kami akan segera menghapusnya.

11. Penyimpanan Lokal

PajakPas menggunakan penyimpanan lokal di perangkat Anda (misalnya preferensi bahasa) untuk meningkatkan pengalaman pengguna. Data ini tidak dikirim ke server kami dan tetap berada di perangkat Anda.

12. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan:

  • Diposting di halaman ini dengan tanggal "Terakhir diperbarui" yang baru
  • Diberitahukan melalui aplikasi untuk perubahan signifikan

Penggunaan aplikasi secara berkelanjutan setelah perubahan merupakan penerimaan terhadap kebijakan yang diperbarui.

Privacy Policy

Last updated: February 9, 2026

Summary: PajakPas collects only the data needed to provide vehicle tax tracking services. We do not sell your data to third parties. We do not display ads. You can delete your account and all data at any time.

1. Introduction

PajakPas ("we", "our", "us", "the app") is a mobile application that helps Indonesian vehicle owners track their vehicle tax (PKB) payments and STNK validity dates. This Privacy Policy explains how we collect, use, store, and protect your personal information when you use our app.

By using PajakPas, you agree to the collection and use of information as described in this policy. If you do not agree with this policy, please do not use our app.

2. Information We Collect

a. Account Information (Required)

When you create an account, we collect:

  • Email address — used for authentication, account recovery, and important service-related communications
  • Name — for your user profile display within the app
  • User ID — automatically generated by our system to identify your account

If you sign in with Google or Facebook, we receive your name and email from those services. We do not receive your password from these services.

b. Phone Number (Optional)

You may optionally add a phone number to your profile. This data is not required to use the app.

c. Vehicle Information (User-Generated Content)

When you add a vehicle, you provide:

  • Vehicle plate number, brand, model, year, color
  • VIN (Vehicle Identification Number) and engine number
  • Owner name and address
  • STNK validity date

d. Tax Payment Information (User-Generated Content)

When you record a tax payment, you provide:

  • Tax year, payment date, and validity period
  • Tax amounts (PKB, SWDKLLJ, admin fees, penalties)
  • Receipt number

e. Document Images (Processed Ephemerally)

When you use the document scanning feature:

  • Photos of your STNK or SSPD tax slip are captured from your camera or gallery
  • Images are sent to a third-party service (Anthropic) for AI-powered text extraction (OCR)
  • Images are processed in real-time and are not stored on our servers or the third-party's servers
  • Only the extracted text data is saved to your account

Use of this feature is optional — you can enter vehicle data manually instead.

f. Backup Data (Processed Ephemerally)

When you use the backup import/export feature:

  • A backup file is created on your device containing your vehicle and tax data
  • Import files are read to restore data to your account
  • Backup files are processed locally and are not uploaded to our servers

g. Information We Do NOT Collect

PajakPas does not collect:

  • Location data (GPS, approximate location, or precise location)
  • Contacts from your device
  • Messages (SMS, email, or other messages)
  • Browsing history or web activity
  • Financial information (credit cards, bank accounts)
  • Health or biometric information
  • Crash logs or diagnostic data
  • Device identifiers or advertising IDs

5. App Permissions

PajakPas requests the following device permissions:

  • Camera — to scan STNK/SSPD documents using OCR. Only used when you choose to scan.
  • Storage/Photo Gallery — to select document images from your gallery. Only used when you pick an image.
  • Notifications — to send tax deadline reminders. You can disable this at any time through your device settings.

All permissions are optional and requested at the time of use. The app remains functional without granting these permissions.

6. Data Security

We implement industry-standard security measures to protect your data:

  • All data is transmitted over HTTPS/TLS (encryption in transit)
  • Data is stored in encrypted databases (encryption at rest)
  • Authentication uses industry-standard JWT tokens
  • Database access is protected by Row-Level Security (RLS) — you can only access your own data
  • Passwords are hashed using bcrypt and never stored in plain text
  • Supabase API keys are restricted to user-level operations only

While we strive to protect your data, no method of electronic transmission or storage is 100% secure. We cannot guarantee absolute security.

7. Data Retention and Deletion

Retention

  • Account, vehicle, and tax data is retained for as long as your account is active
  • Document images are processed in real-time and not stored (zero retention)
  • Backup files exist only on your device and are not stored on our servers

Deletion

  • You can delete individual vehicles and tax records at any time through the app
  • You can delete your entire account via the Profile tab in the app or through our account deletion page
  • When your account is deleted, all data is permanently and immediately removed from our servers
  • There is no recovery period — once deleted, your data cannot be restored
  • Deleted data includes: user profile, all vehicles, all tax records, notification settings, and authentication credentials

8. International Data Transfers

Your data may be processed on servers located outside Indonesia, including:

  • Supabase — servers in the Southeast Asia region (AWS)
  • Anthropic — servers in the United States (for ephemeral OCR processing only)

We ensure that all third-party service providers maintain adequate data protection measures.

9. Your Rights

You have the following rights regarding your data:

  • Access — all your data is visible within the app at any time
  • Portability — use the Backup Export feature in Profile to download all your data
  • Deletion — delete individual records or your entire account from the Profile tab, or via our account deletion page
  • Correction — edit vehicle and tax information at any time through the app
  • Withdraw consent — you can stop using optional features (OCR, backup) at any time

To exercise your rights or if you have questions, contact us at support@pajakpas.com. We will respond within 48 hours.

10. Children's Privacy

PajakPas is not intended for use by children under 13. We do not knowingly collect personal information from children under 13. If you believe a child has provided us with personal data, please contact us and we will promptly delete it.

11. Local Storage

PajakPas uses local storage on your device (such as language preferences) to improve the user experience. This data is not transmitted to our servers and remains on your device.

12. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will:

  • Be posted on this page with an updated "Last updated" date
  • Be notified through the app for significant changes

Continued use of the app after changes constitutes acceptance of the updated policy.
